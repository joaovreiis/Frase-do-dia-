# Bem vindo ao sistema de frases!

# O sistema consiste em exibir uma frase de acordo com o dia do mês
# Dependendo da hora que o sistema está sendo acessado altera automáticamente o papel de fundo.
# Também o comprimento entre "Bom dia!", "Boa tarde!" e "Boa noite!".
# No footer da interface também é exibido a data (DD/MM/AAAA) em tempo real.

# OBS: Todo o conteúdo da página está sendo carregado pelo javascript.
